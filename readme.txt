这是一个批量处理表格类pdf的代码实现
主要思想是利用百度OCR识别文字内容，利用skimage包中提供的形态学操作识别表格位置，从而做到将对应位置的文字内容识别并整理出来
该程序适用于pdf虽然是表格但为扫描版，无法利用pdf解析软件进行处理的情况
